# Instagram Dumper

Currently dumps followed accounts

## Usage:

```
usage: igdump.py [...]

options:
  -h, --help            show this help message and exit
  -s SESSION_ID, --session-id SESSION_ID
                        Session ID
  -i USER_ID, --user-id USER_ID
                        User ID
  -n USER_NAME, --user-name USER_NAME
                        User name
````
